# 真实 EPUB 交互测试计划

## 1. 目标

本文档用于定义基于本机真实 `.epub` 样本的 demo 手工验收范围，重点验证阅读器在真实用户操作下的稳定性、一致性与可预期性。

本轮优先覆盖以下交互能力：

- 打开 EPUB
- 初始渲染与元信息显示
- 滚动阅读
- 分页阅读与翻页
- 目录跳转
- 搜索
- 点击搜索结果跳转
- 字体大小调整
- 阅读模式切换
- 主题切换
- 跨章节定位与当前位置保持

## 2. 测试范围

本计划面向 `packages/demo` 中的浏览器 demo，不直接替代单元测试、集成测试或 Playwright 自动化测试。

本计划重点关注：

- 真实书籍打开后是否能进入可读状态
- `canvas / dom / hybrid` 三类章节路径在真实交互下是否一致
- 用户连续操作后，定位、进度、目录、搜索结果、字体设置是否保持正确
- 已发现的复杂出版社样式书籍是否不再出现明显错位、跳转失效或模式切换异常

本计划暂不覆盖：

- PDF、MOBI 等非 EPUB 格式
- 触摸手势
- 浏览器刷新后的持久化恢复
- 书签、批注、选区高亮

## 5. 验收标准

若某个用例未特别说明，统一满足以下预期：

- 操作后无崩溃、空白页、永久 loading、控制台致命错误
- 当前章节、页码、总页数、书名等元信息能同步更新
- 页面位置变化符合用户直觉，没有明显跳错章、跳错页或回弹
- 同一操作连续执行 2 到 3 次，结果保持一致
- 切换书籍或切换模式后，不应残留上一本书的目录、搜索结果或分页状态

## 6. 核心交互用例

### T01. 打开 EPUB 并完成初始渲染

样本：

- `S1`
- `S2`
- `S3`
- `S4`

步骤：

1. 打开 demo 页面
2. 依次选择一本 EPUB
3. 等待首屏内容显示

预期结果：

- 书名、章节信息、页码信息出现
- 首屏正文可见，不是空白区域
- 没有明显资源丢失导致的大面积错版
- 当前渲染模式可解释
  - 简单章节允许为 `canvas`
  - 复杂出版社样式章节允许为 `dom`

### T02. 滚动模式下向下滚动与向上回滚

样本：

- `S1`
- `S2`

步骤：

1. 保持阅读模式为 `scroll`
2. 连续向下滚动约 3 到 5 屏
3. 记录当前章节与页码
4. 再向上滚动约 2 屏

预期结果：

- 内容连续滚动，无抖动和大段空白
- 页码与章节信息随滚动更新
- 跨章节时，当前章节切换正确
- 回滚后位置接近预期，不发生突然跳回首章或跳到错误章节

### T03. 分页模式下翻页

样本：

- `S2`
- `S3`

步骤：

1. 切换到 `paginated`
2. 连续执行下一页 5 次
3. 连续执行上一页 3 次
4. 到章节边界时继续翻页

预期结果：

- 每次翻页只前进或后退一个稳定页步
- 页码单调变化，不重复、不跳号过大
- 到达章节末尾后可进入下一章节
- 反向翻页时可回到上一章节
- 没有出现内容重叠、页面只渲染半屏、页码已变但正文未变

### T04. 滚动模式与分页模式切换时保持阅读位置

样本：

- `S1`
- `S4`

步骤：

1. 在 `scroll` 模式下阅读到中部位置
2. 记录当前章节与页码或进度
3. 切换到 `paginated`
4. 再切回 `scroll`

预期结果：

- 两次切换后仍留在原章节附近
- 不应回到本章开头或全书开头
- 字体、主题、当前书籍状态不丢失
- 若当前位置在 `canvas/dom` 邻接边界，切换后仍能正确定位

### T05. 点击目录项跳转

- 每次点击都跳到对应章节，不跳错
- 当前目录高亮随位置变化更新
- 跳转后书名、章节、页码信息同步变化
- 若目标章节为复杂样式章节，应仍能正常显示

- 不会一直停留在旧章节高亮

### T07. 搜索存在结果的关键字

- `S1`：`债务`

- `S4`：`Rust`

步骤：

1. 输入关键字

3. 观察结果列表

预期结果：

- 搜索在合理时间内返回结果
- 结果列表中的 `href` 与摘录可读
- 多章节命中时，结果数量与分布看起来合理
- 搜索期间不会卡死 UI

样本：

- `S1`
- `S4`

步骤：

1. 在当前章节记录页码与位置
2. 将字体从默认值增大两档

- 页数允许变化，但当前位置不应错章
- 复杂章节不应因字体变化出现明显遮挡、重叠或空白大片

- 背景色与文字色同步变化
- 文本仍可读，对比度可接受
- 主题切换不应导致重新打开书籍或丢失当前位置

- 不残留上一本书的章节标题或搜索结果
- 新书仍能正常执行滚动、分页、搜索

- `S1`

- 不应出现大面积文字重叠、异常断裂、首字漂移到错误位置
